Overview

Women Safety and Cyber Protection System is a safety-focused application designed to help women during emergencies and improve online security awareness. The platform provides emergency assistance, incident reporting, location-based safety features, and cyber safety guidance.

Features
Emergency SOS Alert
Live Location Sharing
Incident Reporting System
Nearby Police Station and Hospital Locator
Cyber Safety Awareness Module
User Authentication and Secure Login
Community Safety Alerts
Technologies Used

Cloud Services
Project Architecture
Frontend  →  Backend  →  Firebase Database
    ↓             ↓
Android App  →  Cloud Services
How It Works
User registers and logs into the application.
Emergency contacts are added.
SOS button sends emergency alerts and location details.
Users can report unsafe incidents.
Reports are stored securely in Firebase.
Users receive safety alerts and access cyber safety resources.
Installation
git clone https://github.com/Athikesavan05/women-safety-system.git
cd women-safety-system

Configure Firebase credentials and run the application.
